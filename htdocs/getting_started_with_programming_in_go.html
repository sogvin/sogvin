Getting started with programming in Go

Pick a text editor

Programming involves writing code, you do this with an text editor. There are plenty to choose from. I recommend you choose one of the editors listed below depending on your platform. Though it's like picking out your first bicycle, it should be your decision. Please do a internet search for text-editors and try out others aswell. The important thing is, it has to be a text editor, not a word processor like Word.

  • GEdit - my recommendation if you've never used one before
  • Atom
  • Notepad++
  • Sublime
Download Go

Right, the editor up and running. Time to install the Go language and its standard packages. Get the latest version from https://golang.org/dl . Follow the installation instructions carefully and make sure it works. In this process you will need to use a terminal. The terminal is where you will enter commands for the computer to execute. Install the language now, it shouldn't take more than a few minutes.
